# 🌱 Agro Forte, Futuro Sustentável

Projeto desenvolvido para o Concurso de Programação promovido pelo SENAR-PR e SEED-PR.

## 📖 Sobre o Projeto

Este projeto apresenta uma página inicial moderna e responsiva com o tema:

> **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**

O objetivo é destacar a importância da sustentabilidade, da tecnologia no campo e da educação para o desenvolvimento do agronegócio brasileiro.

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Responsividade com Media Query

---

## 🎨 Funcionalidades

- Página inicial moderna
- Layout responsivo
- Tema sustentável
- Navegação simples
- Cards informativos
- Design com cores relacionadas ao meio ambiente
